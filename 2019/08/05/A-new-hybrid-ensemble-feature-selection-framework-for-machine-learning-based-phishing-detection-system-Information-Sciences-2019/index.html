A new hybrid ensemble feature selection framework for machine learning-based phishing detection system（Information Sciences 2019）

发表于 2019-08-05 | 分类于 CCF-B |

一作：Kang Leng Chiew
研究所：马来西亚沙捞越大学计算机科学与信息技术学院
主页：http://www.archive.unimas.my/faculties/fcsit/chiew-kang-leng.html

2 概述

  本文为基于机器学习的钓鱼网站检测方法提出了一个新的特征选择框架，叫做Hybrid Ensemble Feature Selection (HEFS)。
  （1）在HEFS的第一阶段，提出了一种新的累积分布函数梯度(CDF-g)算法来生成主要特征子集，然后将这些特征子集输入到数据扰动集成中来生成次要特征子集。
  （2）第二阶段通过使用函数扰动集成从次要特征子集导出一组基线特征。
  （3）总体实验结果表明，当HEFS与随机森林分类器集成时表现最好，其中基线特征正确地区分了94.6％的网络钓鱼和合法网站，只使用了20.8％的原始特征。
  （4）另一个实验表明，在随机森林中使用基线特征(总共10个)优于在SVM、朴素贝叶斯、C4.5、JRip和部分分类器中使用所有特征(总共48个)。

数据集（特征）下载：https://data.mendeley.com/datasets/h3cgnj8hft/1



3 介绍

3.1 特征选择的两个主要技术

（1）Wrapper
  Wrapper方法迭代运行，每次运行都生成一个特征子集，并通过分类对其进行评估（即训练和测试)。当评估一组较大的特征时，迭代次数将以指数形式增长，对于实时应用程序在计算上变得不切实际。
（2）Filter measure
  Filter measure是根据统计和信息理论计算出来的度量，它可以反映每个特征的优点，而不需要考虑任何特定的分类。
  因此，在本文中关注 Filter measure，其计算密集低，更适用于特征选择。

3.2 本文采用技术

  • CDF-g算法自动标识特征 cut-off rank，从而得到一组高效和紧凑的基线特性。
  • 为了进一步提高基线特征的稳定性以及减少过拟合，将CDF-g算法扩展为由数据扰动和函数扰动技术组成的混合集成结构。
  • 数据扰动需要将数据集细分为多个分区，而函数扰动则涉及对相同的数据集应用不同的过滤措施，将完整的特性选择框架称为混合集成特性选择(HEFS)。

3.3 本文贡献

（1）利用一种新的CDF-g算法，使用混合集成结构生成稳定和健壮的特性子集；
（2）在减少机器处理资源的同时，实现具有竞争力的检测精度；
（3）通过使用完全可计算的技术自动选择最有效的网络钓鱼特征，消除了人为干预的需要；
（4）提供一个灵活和健壮的功能选择框架，可广泛应用于不同的数据集;
（5）应用所提出的特征选择框架和推荐的最佳性能分类器，加速开发准确高效的基于机器学习的钓鱼检测系统;
（6）规定基线特征（即最佳特征子集）作为反网络钓鱼研究人员的实际标准，以对新提出的机器学习技术进行基准测试；
（7）构建了一个大型的可公开访问的机器学习数据集，与大多数反钓鱼研究高度相关。

4 特征选择框架



4.1 混合集成策略

（1）对于数据集分区 $P_j$，过滤度量 ${FM}_k$ 基于该分区中的原始特征值计算过滤度量值，对过滤度量值进行排序并传递到下一阶段；

（2）然后应用CDF-g算法以产生特征截止等级（feature cut-off rank）$\tau_{j,k}$；

（3）对所有数据集分区 $j = 1,2,\dots,J$ ，使用过滤度量 $FM_k$ 分别为每个分区创建过滤度量值，使用CDF-g算法最终得到特征截止等级列表 {$\tau_{1,k}, \tau_{2,k}, \dots, \tau_{J,k}$}；

（4）使用式(1)得到特征截止等级的平均值，即$\bar{τ}_k$，引用{$P_1, P_2, …, P_J$}，以生成主要特征子集{$FS_{1,k}, FS_{2,k}, …, FS_{J,k}$}。 具体地，每个主要特征子集$FS_{j,k}$由$P_j$中的 $top-\bar{τ}_k$个特征组成，其根据过滤度量值排序；




（5）数据扰动集成部分将对主特征子集数组进行求交运算，如式(2)所示，生成子特征子集${FS}_k$。在这个交集操作中，只保留主要特征子集中常见的特征。通过考虑所有分区的一致性，可以帮助丢弃某些分区中可能存在的任何不规则特性。一个真正具有预测性的特性最有可能存在于所有数据集分区中，因此可以(通过交集操作)将其作为子特性子集的一部分进行选择。这就得出了一个单一的数据扰动周期；



（6）函数扰动集成部分通过联合运算将其输入进行集成，如式(3)所示，得到最佳特征子集，即，基线特征集。通过函数扰动周期，可以利用不同过滤度量的智能，从而得到不易过拟合的基线特征集。


4.2 CDF-g算法 —— 自动标识 Feature cut-off rank

Feature cut-off rank：被定义为filter measure values有序列表中的特定阈值rank，其中位于截止等级之外的特征被认为是不相关的并因此被丢弃。图2展示了截止等级$τ$，其中所选择的特征子集被不连续的矩形包围。




（1）离散随机变量 $X$ 取某一特定值 $x$ 的概率一般称为概率密度函数 $P (X = x )$；
（2）随机变量 $X$ 的CDF（Cumulative Distribution Function）定义为 $F_X(t) = P (X \le t)$；
（3）在特征选择中采用CDF，假设离散随机变量表示每个特征的过滤度量值，利用CDF曲线来可视化过滤度量值的累积分布。通过分析CDF曲线的梯度变化，可以在过滤度量值范围内识别出高原区域，高原区域是具有零梯度的曲线的部分，这表明连续值具有相对较大的间隙。过滤度量值的CDF曲线中的一个高原区域表示两个特征子集之间的分离，这两个子集在预测能力方面存在显著差异。因此，本文利用这个概念来确定排名最靠前的特性的截止排名。将上述CDF梯度方法称为CDF-g；
（4）为了计算CDF曲线上任意点 $R_i$ 处的梯度，对内部点采用中心差分，边界点采用单边(正向或反向)差分，如式(6)所示；

6 实验评估

（1）评估1 — 各种分类器在不同特征子集（从过滤度量排名中得到）上的性能
  通过比较分类器之间的精度稳定状态，Random Forest的准确率高于其他分类器，达到95%以上。









（2）评估2 — 选定分类器的过滤度量的适应性
  信息增益，卡方和对称不确定性是与随机森林分类器结合使用的前3种最适合的过滤措施。



（3）评估3 — 基线特性的性能
  最后的基线特性集包含10个特性，如表1所示。评估结果如表2所示，评估结果验证了所提出的HEFS框架的有效性，该框架产生了一组非常有效的基线特性。NumDots, UrlLength, AtSymbol, NoHttps , and IpAddress这些以前认为重要的特征都过时了，这是由于近年来网络钓鱼趋势的演变，钓鱼者们使用新的方案来逃避检测。因此，应该停止使用这些过时的特征。





（4）评估4 — 运行时间分析比较
  评估的目的是根据现有的特性选择技术（(即过滤度量)）对HEFS的运行时性能进行基准测试。表3为对每个样本的平均分类时间(即，测试时间)，其中评估1500个钓鱼和1500个合法数据样本。结果表明，使用HEFS生成的基线特征集测试时间最低，这是合理的，因为减少特征的数量也会降低分类器的计算复杂度。因此，所提出的HEFS框架在处理大型数据集和实时应用程序时具有一定的优势。



（5）评估5 — 在其他机器学习数据集进行基准测试
  为了进一步验证该技术的有效性，HEFS框架以Hadi等人[11]最新的钓鱼检测技术进行基准测试。Hadi等人提出了一种新的分类算法——快速联想分类算法(FACA)，并在UCI机器学习知识库[17]的一个大型钓鱼数据集上进行了评估。表4所示的基准测试结果表明HEFS是健壮的，可以灵活地适应不同的数据集。
