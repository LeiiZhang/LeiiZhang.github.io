2 概述

  本文提出并构建了一个可下载、通用、全面的大型标准离线数据集，包含15000个钓鱼网页和15000个合法网页。数据集下载：http://www.fcsit.unimas.my/research/legit-phish-set



Online phishing is a security threat which combines social engineering and website spoofing techniques to deceive users into revealing their confidential information

3 提高数据集质量的影响因素

  为了建立一个适用于大范围实验的标准离线数据集，考虑以下因素：
（1）大小
  表4中列出的数据集的期望值为28,521。因此，我们建议在本文中构建包含30000个样本的标准离线数据集，分别包含15000个网络钓鱼和15000个合法网页。
（2）网站类别的分布
  对于合法网页数据集，只参考排名靠前的列表(例如Alexa排名前100万的列表)，对网页的分类和分布比例强调较少。
  对于钓鱼网页数据集，对表1中2012-2014钓鱼攻击的每个类别进行平均，得到钓鱼数据集的新分布比例（最后一列）。
（3）语言种类
  数据集包括一些通用世界语言，如英语，法语，日语，中文，印地语，韩语。
（4）各种各样的资源
  如表3所示，数据集中包含的重要元素是：URL、HTML、CSS、Favicon、Image resources、SVG、JavaScript、Web font files、Screenshot、WHOIS information、Other uncommon file type。












4 数据集构建步骤



（1）数据源选择
  1）Legitimate：从Alexa 前一百万的列表中选择14500个url作为高人气网站样本，而从DMOZ和BOTW中选择500个url作为低人气网站样本。
  2）Phishing：在“valid phishes”和“online”类别下，从PhishTank中选择15,000个url作为钓鱼爬虫的输入。
（2）爬虫器构建
  1）WGET-获取网页：https://www.gnu.org/software/wget/
  2）WebShot-获取网页截图：http://www.websitescreenshots.com/
  3）WHOIS-获取域名和IP地址的注册记录：https://docs.microsoft.com/en-us/sysinternals/downloads/whois
（3）数据集标记和聚合
  下载过程从2016年3月20日开始，至2016年4月30日结束。爬虫程序将下载表3中的所有的资源，保存在指定的文件夹中，即每个合法网页的文件夹名称为L00001至L15000，每个钓鱼网页的文件夹名称为P00001至P15000。每个指定文件夹包含5个子文件夹，即RAW-HTML、SCREEN-SHOT、URL、WEBPAGE和WHOIS。
（4）数据集细化
  1）确保每个下载网页的内容完整；
  2）删除损坏的元素和所有多媒体文件；
  3）平衡网页类别的分布。
