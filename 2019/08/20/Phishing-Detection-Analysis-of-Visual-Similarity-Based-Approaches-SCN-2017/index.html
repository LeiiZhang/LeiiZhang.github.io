2 概述

  基于视觉相似性的钓鱼检测技术利用文本内容、文本格式、HTML tags、层叠样式表(CSS)、图像等特征集来做出决策。这些方法通过使用各种特征将可疑网站与相应的合法网站进行比较，如果相似性大于预先定义的阈值，则被分为网络钓鱼。本文综合分析了钓鱼攻击的特点、钓鱼攻击的利用、基于视觉相似性的钓鱼攻击的检测方法及其比较。

  视觉相似性方法的优点：
（1）为了避免钓鱼检测技术，攻击者通常会插入 images、Flash、ActiveX 和 Java Applet 来代替 HTML 文本。基于视觉相似度的检测方法可以快速检测出网络钓鱼网页中存在的嵌入对象。
（2）基于视觉相似性的技术利用 signature 识别钓鱼网页，signature 是通过去掉整个网站的共同特征来创建的。因此，一个 signature 足以检测单个网站或网站不同版本的各种目标网页。

4 基于视觉相似性的钓鱼检测与过滤分类

  钓鱼检测技术广泛分为两类：（1）基于用户教育；（2）基于软件。
  基于软件的技术进一步分为：（1）黑名单；（2）启发式；（3）基于视觉相似性。
  大致将基于视觉相似性的方法分为：（1）HTML文档对象模型(DOM)树；（2）视觉特性；（3）层叠样式表(CSS)相似性；（4）基于像素的；（5）视觉感知；（6）混合方法。如图 7 所示。



4.1 文档对象模型（DOM）

  DOM 是一种独立于语言的多平台公约，用于在 XML、XHTML 和 HTML 文档中解释对象。DOM 表示文档的逻辑结构，处理和控制文档的方式。在这个模型中，文档以树的形式表示，因此称为 DOM 树，如图 8 所示。在基于 DOM 的钓鱼检测系统中，将可疑网页的 DOM 树与合法网页进行比较。



  Rosiello 等人[48] 提出了一种在多个网站上重用相同信息的方法。如果用户重用相同的信息（即，相同的用户名，密码等），系统将生成一个警告。系统比较了数据最初输入的第一个页面和数据重用的另一个页面的 DOM 树，如果发现这两个页面之间的 DOM 树相似，则将视为钓鱼攻击或其他攻击；否则为合法重用，如图 9 所示。该方法保留一个以前访问过的网站列表，以及表单中填写的信息，以便与当前的网站信息进行比较。这种方法的缺点是，它只比较新网页和以前访问过的网页。



  网页的文档对象模型由树表示。树是图 G(V, E) 的一种类型，其中任意两个顶点之间只有一条路径。顶点表示 tags，边表示 tags 之间的层次结构。两个 DOM 树 $T(V, E)$ 和 $T’(V’, E’)$ 在满足以下属性时等价：
  （1）对应顶点的值相同；
  （2）匹配边的索引相同；
  （3）两棵树同构 [65]。

4.2 视觉特征

  有两种类型的视觉特征：（1）文本内容；（2）基于文本的特征（如字体颜色、字体大小、背景颜色、字体族等）。用于比较的视觉特征如表 1 所示。以下介绍基于视觉特征的解决方法：



（1）视觉相似度评估
  Liu 等[66] 提出了一种包含两个模块的方法。第一个模块检测本地电子邮件服务器上的可疑 url 和关键字。在检测到电子邮件中的可疑关键字或 url 之后，第二个模块比较可疑网页的块级别、布局和样式相似性。图 10 显示了该方法的体系结构。



  • Block Level：将可疑网页划分为块，对每个块提取视觉特征（表 1 所示特征）与对应的合法网页进行匹配，然后系统通过匹配块的加权平均计算块级相似性；
  • Layout Level：将匹配块的总数与合法网页中存在的块的总数进行分块来计算。
  • Overall Style：通过提取每个网页的风格特征直方图来计算。当给定网页与合法网页之间的相似度大于某个预定义的阈值时，系统将该给定网页标记为钓鱼网页。

[66] W. Liu, X. Deng,G.Huang,and A. Y. Fu,“An antiphishing strategy based on visual similarity assessment,” IEEE Internet Computing,vol.10, no.2,pp. 58–65, 2006.

（2）站点签名
  Huang 等[67] 提出了一种创建独特的基于网站签名来识别合法网站的技术，网站签名是使用网站的文本（关键词）和图片创建的。当用户打开一个新网页时，系统将当前打开的网页的签名与数据库中存储的签名进行匹配。如果签名匹配但域名不同，则该网页为钓鱼网页。该方法的性质：

1）系统利用文字和图像特征为网站建立独特的签名。文本特性包括标题关键字、URL关键字和使用 TF-IDF 算法得到的关键字，图像从主页以及所有与主页相关的网页中通过超链接提取；
2）签名是通过从整个网站而不是单个网页中提取特征来创建的，因此，只有一个签名足以检测各种有针对性的假网页的网站；
3）签名使用整个网站的标准特征，因此可以检测部分抄袭合法网站的钓鱼网页；
4）签名是使用图像和文本特征创建的。因此，如果攻击者只在虚假网页中使用图像，该方法也可以检测到这种类型的攻击。

  该方法最初为新访问的网站创建签名并将其与存储的签名匹配。 如果用户第一次访问网站并且其对应的合法站点不存在（即，签名与任何存储的签名不匹配），则系统创建签名并将其存储在数据库中。

[67] C.-Y. Huang, S.-P. Ma, W.-L. Yeh, C.-Y. Lin, and C.-T. Liu, “Mitigate web phishing using site signatures,” in Proceedings of the IEEE Region 10 Conference TENCON, pp. 803–808, November 2010.

4.3 CSS 相似性

  CSS 是一种语言，用于描述文档的格式和设置用 HTML、XHTML 和 XML 编写的网页的视觉外观，设计网页内容，如字体、颜色和页面布局。

（1）BaitAlarm
  Mao 等[71] 提出了比较可疑网站和合法网站的 CSS 相似性的算法。CSS包含一系列规则，这些规则将一组选择器、属性和值与一组声明关联起来。选择器可以分为不同的类别，例如类选择器、tag 选择器、id 选择器和一些属性选择器。属性说明与选择器选择的元素相关的属性。例如，任何标记都包含颜色、边距、字体大小、字体族、填充属性、边框和与这些属性关联的值。通过提取选择器、属性和值，可以确定两个 CSS 的相似性。除此之外，Mishra 和 Gupta[69] 提出了一种基于 URI 和 CSS 匹配的混合解决方案，其中 CSS 匹配是从 BaitAlarm 中借来的，Algorithm 1 用于比较两个网页之间的 CSS。



[71] J. Mao, P. Li, K. Li, T. Wei, and Z. Liang, “BaitAlarm: detecting phishing sites using similarity in fundamental visual features,” in Proceedings of the 5th IEEE International Conference on Intelligent Networking and Collaborative Systems (INCoS ’13),pp. 790–795, Xi’an, China, September 2013.
[69] A. Mishra and B. B. Gupta, “Hybrid solution to detect and filter zero-day phishing attacks,” in Proceedings of the Emerging Research in Computing, Information, Communication and Applications (ERCICA’14), Bangalore, India, August2014.

4.4 基于像素的技术

  基于像素或图像处理的方法使用图像处理应用程序来检查两个网页之间的相似性。基于图像处理的方法要么截屏，要么从可疑网页中提取图像，并将图像与合法网页进行比较。这些方法认为不同组织的两个不同网站不可能是相似的。如果不同站点的两个图像相似，并且它们的 url 不同，那么一个站点就被认为是网络钓鱼，如图 12 所示。



（1）Discriminative Keypoint Features
  Chen 等[72] 提出了一种基于区分关键点特征的反钓鱼方法。该系统包括两个模块。在第一个模块中，系统对可疑网页进行快照，并利用 Contrast Context Histogram (CCH) 在判别关键点周围捕获不变信息；在第二个模块中，系统根据匹配的 CCH 描述符计算两个页面之间的相似度评分。已验证的页面存储在本地数据库中。
  为了确定两幅图像是否相同，该方法从每幅图像中提取重要特征向量，并计算这些向量之间的距离，距离衡量两个图像之间的视觉差异的程度。如果攻击者将文本内容替换为图像，那么这种技术就可以检测到它。

[72] K.-T. Chen, J.-Y. Chen, C.-R. Huang, and C.-S. Chen, “Fighting phishing with discriminative keypoint features,” IEEE Internet Computing,vol.13, no.3,pp. 56–63, 2009.

（2）Goldphish
  Dunlop 等[73] 提出了一种基于浏览器的插件 goldphish 来识别钓鱼网站。Goldphish 利用网站 logo 来识别虚假网站，分为三个阶段：

1）标识提取：Goldphish 从可疑网站中提取网站 logo，然后使用光学字符识别（OCR）软件将其转换为文本；
2）合法网站提取：文本用作“谷歌”搜索引擎的查询。
3）将可疑网站与搜索引擎的 top 结果进行比较。如果结果中有域名与当前网站匹配，则为合法网页；否则为钓鱼网页。

  Goldphish 可以快速识别 popular organizations 的虚假网站。然而，系统的精度取决于 OCR 软件。Goldphish 还保护用户免受“zero-hour”钓鱼攻击。

[73] M. Dunlop,S.Groat,and D. Shelly,“GoldPhish: using images for content-based phishing analysis,” in Proceedings of the 5th International Conference on Internet Monitoring and Protection (ICIMP ’10), pp. 123–128, Barcelona, Spain, May 2010.

（3）Dynamic Watermarking（动态水印）
  Singh 等[74] 使用动态水印技术保护客户端用户，并在攻击过程中抵御攻击者。该技术要求在网站上注册时需提供额外的信息，如密钥、水印图像及其位置。用户通过检查唯一的图像及其位置来确定网站的合法性。包括三个主要部分:

1）Registration：当用户第一次在网站上注册时，通常网站需要用户的凭证，如用户名、密码等。在该技术中，用户还需要输入三个 fields：水印图像、水印位置和密钥，密钥充当数据库的主密钥。
2）Login Verification：当用户注册后打开网站时，他/她必须验证证书。在登录验证过程中，用户必须输入正确的密钥。如果本网站显示正确的水印图像及其位置(与注册时输入的相同)，则本网站是合法的。
3）Closing：用户只需要知道水印图像及其位置。在注销时，用户可以更改新的水印图像和位置。

[74] A. P. Singh, V. Kumar, S. Sengar, and M.Wairiya, “Detection and prevention ofphishing attack using dynamic watermarking,” in Information Technology and Mobile Communication,Communications in Computer and Information Science, pp. 132–137, Springer, 2011.

（5）Earth Mover’s Distance (EMD)
  Fu 等[77] 利用 EMD 来比较两个网页之间的视觉相似性。该技术将网页转换为低分辨率图像（100×100像素的图像），然后利用图像的颜色特征和图像位置的质心进行独特的签名。每个像素由ARGB（alpha、红色、绿色和蓝色）组成。该方法只能检测到在一定阈值水平上与合法网页相似的钓鱼网页。如果网络钓鱼网页与合法网页有部分相似之处，则该技术无法检测到。该方法定期从站点监视服务器更新钓鱼数据库。

[77] A. Y. Fu, W. Liu, and X. Deng, “Detecting phishing web pages with visual similarity assessment based on Earth Mover’s Distance (EMD),” IEEE Transactions on Dependable and Secure Computing,vol.3,no. 4, pp.301–311, 2006.

4.5 使用 Gestalt 理论的基于视觉感知的方法

（1）Gestalt Theory for Visual Perception
  Gestalt Laws of Organization 研究了人们如何感知视觉成分。Gestalt 是一个心理学名词，意思是“统一的整体”。在这个理论中，有六个主要因素决定了视觉系统如何自动地将元素分组成模式，分别为：接近（Proximity）、相似（Similarity）、封闭（Closure）、对称（Symmetry）、共同性（Common Fate）以及连续性（Continuity）。

（2）检测视觉上相似的网页
  Chen 等人[84] 提出了一种利用 Gestalt 理论来识别两个网页之间视觉相似性的方法。作者利用超信号的概念将网页作为独立的单元，并利用算法复杂度理论对这些特殊的超信号进行了比较。为了测试该方法的性能，作者对 12 个真实和 12 个钓鱼网页进行了测试，发现所有的 12 对都被成功地配对为最相似的一对。

4.6 混合方法

  混合方法[79, 81-83] 是两种或两种以上技术的组合，使用从网页中提取的各种特性的组合，如文本、图像、超链接等。混合方法的 accuracy 优于单一特征方法。

（1）贝叶斯方法



  Zhang 等[79] 提出了一种贝叶斯模型来确定可疑网页和合法网页之间的相似度阈值。该方法基于网页的视觉和文本特征，如图15所示。文本内容是网页中出现的 words，articles 和 stop words 除外（如 a、an、the、this、that等）。视觉内容是包括网页布局、图像、logo、表单、背景颜色、字体颜色等在内的特征集合。该方法以下属性：

1）基于朴素贝叶斯规则的文本分类器用于从网页中提取文本；
2）基于 EMD 的图像分类器用于在将网页转换为图像之后处理像素级内容；
3）设置适当的阈值，使用贝叶斯方法离线训练；
4）基于贝叶斯的融合算法用于聚合图像和文本分类的结果。

  该方法有三个主要贡献：

1）提出了一种基于朴素贝叶斯规则的文本分类器用于钓鱼检测；
2）提出了一种确定图像分类器和文本分类器阈值的贝叶斯方法。基于这个阈值，提出的技术可以区分网络钓鱼和合法网页；
3）提出了一种新的贝叶斯分类方法，将文本分类结果与图像分类结果相结合。

[79] H. Zhang, G. Liu, T. W. S. Chow, and W. Liu, “Textual and visual content-based anti-phishing: a Bayesian approach,” IEEE Transactions on Neural Networks,vol.22, no.10, pp.1532–1546, 2011.

（2）混合特征
  Medvet 等人[81] 提出了一种使用文本、图像和网页的整体视觉外观来计算签名的方法。从文本内容、字体大小、前景颜色、背景颜色、字体族及其在网页中的位置提取的文本属性；图像属性包括宽度、高度、src 属性、图像在网页中的位置、颜色直方图[85]及其二维Haar小波变换（二维哈尔小波变换提供了原始图像的低分辨率信息）。
  签名比较。该系统首先比较网站中每个页面中匹配的元素对，文本元素只与文本比较，而图像元素只与图像比较。然后系统计算文本相似度评分、图像相似度评分和总体视觉外观评分。此外，使用这三种评分方法，得到总的相似度评分 $S \in [0,1]$。执行以下步骤来比较合法网页和可疑网页：

（1）取一个可疑网页“$p.$”；
（2）计算可疑网页的签名 $S(p)$；
（3）比较可疑网页的签名 $S(p)$ 与存储的预期合法网页“$p$”的签名，如果签名非常相似，则调用一个警告。

[81] E. Medvet, E. Kirda, and C. Kruegel, “Visual-similarity-based phishing detection,” in Proceedings of the 4th International Conference on Security and Privacy in CommunicationNetworks (SecureComm ’08),245,234 pages, Istanbul, Turkey, September 2008.

5 未决问题和挑战

（1）zero-hour 钓鱼攻击
  大多数反钓鱼技术使用包括 URL、keyword 和 visual appearances 在内的特征集来比较合法站点池中的网站与可疑网站。这些技术需要很大的数据集，但仍然无法检测到 zero-hour 钓鱼攻击。如果攻击者设计了一个新的网页，而其目标（即对应的合法网页）不在数据集中，则无法检测到新的虚假网页（zero-hour attack）。Liu 等[82] 提出了一种检测 zero-hour 钓鱼攻击的技术，但该技术依赖于 TF-IDF 算法和超链接。因此，如何准确检测 zero-hour 钓鱼攻击仍然是一个有待解决的难题。

[82] G. Liu, B. Qiu, and L.Wenyin, “Automatic detection ofphishing target from phishing webpage,” in Proceedings of the 20th International Conference on Pattern Recognition (ICPR ’10),pp. 4153–4156, Istanbul, Turkey, 2010.

（2）语言独立性
  各种文本语言在全球范围内广泛使用，电子商务和银行网站在不同的国家也有不同的文本语言，如亚马逊、eBay 和 Citibank。电子商务和银行站点的布局在不同的语言中几乎是相似的。基于启发式的钓鱼检测技术使用关键字，它们依赖于语言。一些基于视觉特性的技术可以检测到这种攻击，因为它们利用了网页特性，比如 logo、CSS 结构、DOM 树等。只有当钓鱼网站的布局与真实网站相似时，这种技术才会检测到攻击。然而，这些技术无法检测新的网络钓鱼攻击（zero-hour），因为它们将当前网站与存储的数据库进行比较。

（3）网页中嵌入的对象
  攻击者利用图片、JavaScript 等绕过反钓鱼系统。基于图像处理的技术可以检测可疑网页中存在的嵌入对象，因为这些技术对网页进行快照，并将其与相应的合法网页进行比较。但在基于图像处理的解决方案中，识别正确的对应合法的网页是主要问题。基于图像处理的方法也花费了大量的时间来比较可疑的网站和网站池。因此，检测使用嵌入式对象的钓鱼网站仍然是一个开放的挑战。

（4）确定适当的阈值
  阈值是两个网站之间的匹配分数。攻击者构建了一个看起来与合法网站相似的钓鱼网站。如果钓鱼网站从合法网站部分复制（少于50%），那么基于视觉相似性的方法都无法检测到它。因此，调整适当的阈值来检测最大数量的网络钓鱼网站是一项具有挑战性的任务。如果反钓鱼系统提高阈值，则假阴性率增加，如果降低阈值，则假阳性率增加，如图 16 所示。一个好的反钓鱼系统要求假阴性和假阳性率都尽可能的低。
